# 🚗 Car Mileage Predictor

A full-stack web app that predicts car mileage using Machine Learning.

## 🔥 Features

- Predicts mileage based on car details
- Built with Flask (backend)
- Uses LightGBM model
- Simple and clean frontend

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- LightGBM
- HTML, CSS, JavaScript

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
python app.py
```
